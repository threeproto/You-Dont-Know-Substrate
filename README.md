# Substrate区块链开发

本书面向中文开发者，内容包含，
- 区块链技术的基本理解
- Substrate开发指南
  - 存证功能模块开发
  - benchmark
  - 链上升级
  - 部署一条独立的链
- ink!智能合约开发
- 介绍Substrate内置的模块
- 生态工具的使用
  - 区块浏览器


## How to

Install mdbook

```shell
cargo install mdbook
```

Start local server,

```shell
mdbook serve
```

Now go to `http://localhost:3000/` for book contents.
